# Homework Knowledge Hint

## Description

占位 Skill 草案：根据课程资料为书面作业题提供知识点提示，但不直接给出答案。

## When To Use

- 用户希望理解题目涉及的知识点。
- 用户需要解题方向、概念提醒或阅读建议。
- 用户明确要求不要直接输出完整答案。

## Workflow

1. 识别题目主题和课程知识点。
2. 给出相关概念、公式或阅读范围。
3. 提供分步骤思考提示。
4. 检查输出中是否包含直接答案，并在必要时改写为提示。

## Constraints

- 不直接给最终答案。
- 不替用户完成完整作业。
- 不编造课程资料中不存在的要求。
